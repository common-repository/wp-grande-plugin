WP Grande Plugin enables you to customize your wordpress installation backend to the core and much more...

== Description ==
WP Grande Plugin is a versatile WordPress plugin that allows you to change or hide a number of options like administration menu, submenu, 'Dashboard', with forwarding to the Manage page. On top of that, you can also perform changes to WordPress login page like change in logo or hyperlink in login page of your site. This plugin also facilitates you to hide post Meta controls on the Write page and other areas in the wordpress admin area and Write page, so as to simplify the editing interface. 

= A Vast Range of Customization Offer =
You can change (hide or customize) your Dashboard Settings, Admin Bar, Admin Footer items, Login Page, Admin Menu items, Screen Options Metabox items through this plugin. 

= Easy Search Engine Optimization =
Additionally, this plugin is perfect and completes your basic SEO needs like addition of meta script of different search engines like Google, Bing, Yahoo etc for Webmaster tools and Analytics, also provides the facility of xml sitemap Generation, Social Media Buttons and alot more. 

= Requirements =
WordPress version 3.3.2 and later

= What does this plugin do? =
The plugin changes the administration backend and gives you the power to assign rights on a wide range of options. Admins can activate/deactivate each part of the admin menu and even the submenus. Dashboard items can be hidden or changed as per the user's requirements. Wordpress login page can also be modified with this very plugin. User can change logo image, hyperlink from the login page. This plugin is able to customize Screen Options metaboxes and many similar items from menus and submenus. Admin bar also can be hide/show to get much space and simplify your workspace.
Another functionality of this plugin is to do all search engine optimization tasks like services of Webmaster tools and analytics of various popular search engines like Google, Yahoo, Bing etc. This plugin can add all selected social networking sites link to the page, more than that it creates and add XML sitemap to your site. 

= Details =

1.	Visit to Dashboard Page section to hide/show Dashboard heading icon, text, logo, Content logo.  
2.	User can make changes in dashboard welcome panel heading and description.
3.	At the same time it can hide/show all boxes, widgets like recent comments, incoming links, pulgins, quickpress, recent drafts, wordpress blog, wordpress news, etc.
4.	In Admin Bar section it can be used to hide/show upper admin bar along with wordpress logo, screen options, and help menu. 
5.	The Login Page section facilitates to change initial login page items
6.	It can change the WordPress logo and upload your custom image which will show up at your site's login page.
7.	It can change the hyperlink and customize it with any other link of your choice at login page logo area.
8.	Login Logo can be hidden if it is not necessary to show.
9.	Change and hide footer content from site page and version detail through Admin footer section.
10.	In admin menu section, it can rename or remove all menu items like dashboard, posts, media, links, pages, comments, etc. 
11.	Metaboxes section facilitates you to customize post and pages screen option items like excerpt, Discussion, comments, slug, custom field, categories, author, publish, featured image, format etc. 
12.	Removing these items will simplify the page outlook and provide the desired workspace within the add/edit pages.
13.	This plugin provides the basic functions of adding up different webmasters tools codes to your website.
14.	It can be add and remove various social media buttons like twitter, facebook, google plus, linkedin, delicious, reddit, etc.
15.	These buttons can also show up the numbers of visitors who liked/tweet/plused your page or shared it.
16.	It can be specified on which pages the social media buttons are to be shown, such as home, about page, contact page, hello world post, category pages, archives, etc.
17.	Meta codes can be added of google webmaster, bing, yahoo site explorer, Alexa bank, blog catalog, facebook insight to your page's head section
18.	Analytics parameters can be added through this plugin like google analytics, quantcast analytics, clicky, compete, and site meter to your site 
19.	Analytics will assist you to watch the analysis of visitors like visitor place, keyword, traffic stats, etc.
20.	XML sitemap generator enables you to generate the sitemap of your site with a click of a button.
21.	You have the power to set sitemap content, change frequency, and priority of each page in custom way.
22.	Also, users can view the generated sitemap in XML file format with last modified field, and many more.
23. SEO Link Manager allows you to have internal links for better search engine optimization.
24. SEO Image Optimizer ensures you get the best from search engines for your images by appending post title to image title and alt tag.
25. Add a contact form simply by using the shortcode [ContactForm] 

More features coming up..


== Installation ==
1. Unpack the downloaded plugin package.
2. Upload the wp-grande-plugin folder which includes all files to the `/wp-content/plugins/` directory. 
3. Activate the plugin through the `Plugins` menu in WordPress

* or use the automatic install via backend of WordPress


== Frequently Asked Questions ==

= How I can deactivate the plugin =
In the WordPress admin panel go to "Plugins", find "Wp-Grande" plugin and press "Deactivate".
